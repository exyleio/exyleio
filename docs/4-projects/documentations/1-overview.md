# Overview

## Introduction

This is the documentation for the
[exyleio](https://github.com/exyleio/exyleio)
repository.

## Getting started

1. Install NodeJS (16.14 or above) and the yarn CLI
2. Serve documentations website locally
   ```
   yarn start
   ```
3. Build for production
   ```
   yarn build
   ```
